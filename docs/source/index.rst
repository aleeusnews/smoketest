Welcome to Smoketest's documentation!
=====================================

Smoketest is a program that runs tests against websites.

Features:

    * Check status codes

    * Test URLs behind cookie-based auth

    * Check redirect locations

    * Check that HTML text and attributes match simple string patterns

    * Check that responses comply with XML schemas

    * Check that responses comply with JSON schemas

    * Pretty output to your shell

    * Detailed JSON output

    * Log response times

    * Easy to expand input files to include new tests

    * Easy to extend code to add more testing capability

This documentation provides more details on these features, as well as smoketest
internals and how to extend smoketest. For basic usage instructions and
development quickstart, see the README file at the top level of the repository.
